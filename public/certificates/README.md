# Certificate images for Digital Credentials

1. Add your certificate images here (e.g. `aws-cert.jpg`, `react-course.png`).
2. Open `src/components/Projects.tsx` and find the `certificates` array.
3. Add one entry per certificate: `{ id: 2, title: 'Certificate name', image: '/certificates/your-filename.jpg' }`.
4. Use clear, short filenames (no spaces). JPG, PNG, and WebP work best.
